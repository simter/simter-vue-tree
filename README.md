# simter-vue-tree

A vue tree component.

## Requirement

| Name   | Version |
|--------|---------|
| Vue    | 2.5+    |
| Parcel | 1.10+   |

## Develop

```
yarn install
yarn start
```

Then open <http://localhost:1234>.

> Use [parcel] run to `temp` directory.

## Build

```
yarn run build
```

> Use [parcel] build to `dist` directory.

[parcel]: https://parceljs.org
[yarn]: https://yarnpkg.com
[Vue]: https://vuejs.org
